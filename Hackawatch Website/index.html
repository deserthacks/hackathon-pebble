What is Hackawatch?







  1. Hackawatch is a utility application for the Pebble Time
  2. With Hackawatch, users and organizers can remain organized at the click of a button
  3. So far Hackawatch can:
    • Create and personalized "pins" for their Pebble Timeline
    • Set vibrating reminders that would remind you when hacking ends
    • Utilize the accelerometer and magnetometer to have a compass point you to to find the venue (coming soon)
